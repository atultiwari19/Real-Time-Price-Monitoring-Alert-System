# Real-Time-Price-Monitoring-Alert-System
* Streamlit-based UI for real-time data visualization using Fake Store API.
* Developed a real-time price analysis application using Streamlit for interactive visualization.
* Fetched and processed real-time API data using Python Pandas for trend analysis.
* Integrated MySQL database and SQL queries for structured data storage and analysis.
* Created interactive Plotly dashboards with filtering capabilities for dynamic analysis.
